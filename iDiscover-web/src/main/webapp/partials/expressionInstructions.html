Expression must evaluate to true or false.

Use the syntax ${FormVariable.ElementVariable} for variables.
If the variable is on the current form you can use the abbreviated syntax ${ElementVariable}.
Use #number,#other,#string suffixes for lookups.

Example: ${FormVarName.AmountVarName}+${FormVarName.LookupVarName#number}

Special Variables:
${studyEnrollment.dateAdded} = Date participant data initially collected
${studyEnrollment.participantStudyId} = Participant Study ID
${studyEnrollment.secondaryStudyId} = Secondary Study ID
${studyEnrollment.empi} = EMPI
